## 数据结构和算法 DataStructure & Algorithm (C语言实现) 

数据结构和算法，是计算机入门的基础的重中之重，是在完成一门编程语言（例如C语言）后可以掌握的基础课程。由于C语言是一门语法稍微简单，并且多数使用作为编程入门的语言。因此，会作为数据结构和算法实现的语言之一。

**前置需求**：
- C语言中基本数据类型、表达式、变量；
- 分支、循环；
- 函数调用与被调用；
- 数组、指针，字符串，结构体，重定义；
